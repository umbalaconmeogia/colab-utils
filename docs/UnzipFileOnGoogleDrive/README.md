# Unzip a large files on Google Drive

## 1. Overview

[UnzipFileOnGoogleDrive.ipynb](../../src/UnzipFileOnGoogleDrive.ipynb) is a script to unzip large files (more than 100GB for example) on Google Drive, running on Colab.

## 2. Usage summary

1. Download [UnzipFileOnGoogleDrive.ipynb](../../src/UnzipFileOnGoogleDrive.ipynb) to your Colab or Google Drive.
2. Open the script in Colab.
3. Run the script, specify the .zip or .gz file on the Google Drive to be unzip, run the script and wait until all files are unzipped to your Google Drive.
